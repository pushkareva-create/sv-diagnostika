МЕТОДИКА РАСЧЕТА РЕЗУЛЬТАТОВ ОПРОСА

1.	Основа модели

Каждый вариант ответа соответствует одному из четырех управленческих паттернов.

A — Эксперт продаж.  •	Личное участие в достижении результата.  •	Основной подход: «Сделаю сам».  •	Руководитель активно включается в операционную деятельность и опирается прежде всего на собственную экспертизу.

B — Администратор  •	Управление через контроль, правила и дисциплину.  •	Основной подход: «Организую и проконтролирую».  •	Руководитель обеспечивает выполнение задач через регламенты, контроль и управленческую дисциплину.

C — Системный руководитель  •	Управление через процессы и организацию работы.  •	Основной подход: «Изменю систему».  •	Руководитель устраняет причины проблем через изменение процессов и системы регулярного менеджмента.

D — Наставник  •	Управление через развитие сотрудников.  •	Основной подход: «Помогу сотруднику стать сильнее».  •	Руководитель инвестирует время в развитие компетенций и самостоятельности команды.

2.	Механика ответов
3.	
В каждом вопросе участник ранжирует все четыре варианта по степени предпочтения.

|	1 место /  +3 балла |	2 место / +1 балл  |	3 место / - 1 балл |	4 место / - 3 балла  |

Баллы начисляются по каждому варианту ответа в соответствии с выбранным рангом и суммируются по соответствующей шкале управленческого паттерна.

3.	Подсчет профилей
        
Суммируются баллы отдельно по четырем шкалам:

Total A - Эксперт продаж,  Total B - Администратор,  Total C - Системный руководитель,  Total D - Наставник.

Диапазон значений каждой шкалы: Минимум: −36 баллов, Максимум: +36 баллов

Для визуализации результатов показатели дополнительно переводятся в шкалу от 0 до 100 по формуле:P = ((Score + 36) / 72) × 100 ,

где: 0 соответствует минимально выраженному паттерну;  50 соответствует среднему уровню выраженности;  100 соответствует максимально выраженному паттерну.

4.	Определение управленческого профиля

Для построения управленческой карты рассчитываются две координаты.

Первая ось отражает управленческий фокус: от ориентации на задачи и контроль к ориентации на развитие людей и системное управление.

Формула: X = (C + D) − (A + B)

Положительное значение означает усиление системного и развивающего подхода.
Отрицательное значение означает усиление ориентации на личное участие и оперативное достижение результата.

Вторая ось отражает способ управления: от личного участия к управлению через систему.

Формула: Y = (B + C) − (A + D)

Положительное значение означает управление через процессы, правила, контроль и организацию работы.
Отрицательное значение означает управление через личное участие, экспертность и непосредственное влияние руководителя.

Полученные координаты используются для построения визуальной карты управленческого профиля.

5.	Определение типа руководителя
    
Тип руководителя определяется на основании выраженности четырех управленческих паттернов.

Шаг 1. Определение сбалансированного профиляЕсли разница между максимальной и минимальной шкалой составляет не более 6 баллов, руководителю присваивается профиль «Сбалансированный руководитель».Такой результат означает отсутствие выраженных перекосов между управленческими подходами.

Шаг 2. Определение доминирующего профиляЕсли разница между первой и второй по величине шкалой составляет не менее 8 баллов, руководителю присваивается профиль по наиболее выраженной шкале:Эксперт продаж; Администратор; Системный руководитель; Наставник.

Шаг 3. Определение смешанного профиляЕсли разница между первой и второй шкалой менее 8 баллов, руководителю присваивается комбинированный профиль, включающий два наиболее выраженных управленческих паттерна:Эксперт продаж — Администратор;Эксперт продаж — Системный руководитель;Эксперт продаж — Наставник;Администратор — Системный руководитель;Администратор — Наставник;Системный руководитель — Наставник.

6.	Расчет значений управленческих функций
   
Постановка задачи = (A + B) / 2,   Планирование = C,   Инструктаж = (A + D) / 2,   Контроль = B,   Оценка результатов = (B + C) / 2,  Мотивация = D,   Организация работы = C,   Личный пример = A.

Формулы являются экспертной интерпретацией и не измеряют абсолютные значения функций.

7	Слабые функции  

Определяются три функции с минимальными значениями. Они рассматриваются как основные зоны развития руководителя.

8.	Модель Адэра
   
Баланс значений по кругам внимания руководителя рассчитывается из показателей функций.

Task = (Постановка задачи + Планирование + Контроль) / 3, 

Team = (Инструктаж + Организация работы) / 2, 

Individual = (Оценка результатов + Мотивация + Личный пример) / 3.

9.	Рекомендации	
Рекомендации формируются не на основании конкретных ответов, а на основании управленческого профиля и баланса Task, Team и Individual.  То есть два участника с разными ответами, но имеющие одинаковый профиль и близкий баланс модели Адэра, получат одинаковые рекомендации.

10.	Ограничения инструмента
Предлагаемый инструмент диагностики предназначен для:  •	быстрой ориентации руководителей;  •	проблематизации для анализа управленческих привычек;  •	обсуждения перехода от роли сильного исполнителя к роли руководителя;  •	изучения модели Джона Адэра.

Инструмент не является психометрическим, применим в качестве быстрого погружения руководителей продаж в концепцию Д.Адэра Action-Centered Leadership.
